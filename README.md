# Printf pre ESP32
# Nahratie programu
* Spustiť .bat súbor
* V .bat súbore zmeniť COM port
* Pri nahrávaní - pripojení ESPToolu - držať EN, BOOT tlačidlo, pustiť EN.
* Počas nahrávania už je možné BOOT tlačidlo pustiť
* Po nahratí programu stlačiť EN
